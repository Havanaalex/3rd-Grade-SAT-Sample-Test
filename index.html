🚀 How to Go Live on GitHub Pages

1

Create a Repo

Go to github.com, click New Repository, name it something like sat10-grade3, and set it to Public.

2

Upload All 4 Files

Click Add file → Upload files and drag in all four HTML files. Make sure this file is named index.html.

3

Enable Pages

Go to Settings → Pages, set Source to main branch / root, and click Save.

4

Share the Link!

GitHub gives you a URL like yourname.github.io/sat10-grade3. Share it with your school!
